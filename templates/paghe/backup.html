Aiuto — Gestione Backup

Cos'è questa pagina
Qui puoi creare, scaricare e ripristinare i backup del database in formato JSON + ZIP del database SQLite. I backup sono essenziali per la sicurezza dei dati e vengono automaticamente programmati ogni notte alle 03:00.

Come si usa
1. Scegli il tipo di backup: Completo (tutti i dati), o parziale (solo selezioni specifiche).
2. Clicca Crea Backup per generare il salvataggio. Verranno creati sia il file JSON (dati strutturati) sia lo ZIP del database (copia fisica).
3. Nella tabella puoi Scaricare il JSON, lo ZIP del DB, o Ripristinare direttamente il DB cliccando "Ripristina DB".
4. Usa Pulisci backup vecchi per rimuovere automaticamente quelli più vecchi di 6 mesi.
5. Upload e ripristino — carica un file JSON dal tuo computer per ripristinare dati da un backup esterno.

Backup del database (ZIP)
Ogni backup crea automaticamente una copia del file db.sqlite3 tramite l'API sqlite3.backup(), che garantisce una copia consistente anche durante le scritture. Lo ZIP contiene un unico file db.sqlite3 pronto per il ripristino. Dopo un ripristino del DB, riavviare il server per sicurezza.

Avvertenze
Il ripristino sovrascrive i dati esistenti nel database. Si consiglia di fare un backup completo prima di qualsiasi ripristino. I backup automatici notturni sono attivati di default e gestiti dallo scheduler integrato (APScheduler).
